핵심 내용

- 배당 가치 투자 전략의 원리를 이해한다.
- 파이썬 데이터 분석을 배당 가치 투자 전략에 응용한다.
- 투자 결정에 필요한 데이터를 추출한다.

타겟
- 장기 가치 투자로 노후를 대비하고 싶은 직장인
- 주식 투자 포트폴리오를 직접 관리하고 싶은 투자자
- 파이썬 데이터 분석을 어떻게 활용하는지 알고 싶은 일반인


비추천
- 단기 트레이딩으로 고수익을 원하는 투자자
- 내가 직접하는 거보다 다른 사람에게 위임하기를 좋아하는 분

주의사항
- 윈도우 기반 파이썬 데이터 분석

홍보 문구 -> 역본능 투자법
- 본능을 거스를 수 없다면 속여라!
- 땅을 파면 돌이 나오고
- 데이터를 파면 돈이 나온다!

문제
- 장기 가치 투자를 원한다. 그런데 어떻게 시작할지 모른다.
- 데이터 분석으로 종목을 선정하고 매수 타이밍을 확인할 수 있다.
